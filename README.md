# Ademi - E-commerce App for Souvenirs & Art 🇰🇿

**Ademi** is a modern iOS application designed for a premium souvenir and jewelry store. The project focuses on creating a seamless shopping experience using clean UI/UX and real-time cloud integration.

## 🚀 Key Features
- **Multi-method Authentication:** Secure login via Google, Apple ID, and Phone Number (Firebase Auth).
- **Dynamic Catalog:** Browse a curated collection of 40+ unique art panels, souvenirs, and jewelry.
- **Real-time Backend:** Fully integrated with Google Firebase for data synchronization and user management.
- **Modern Architecture:** Built using the **MVVM** pattern for scalability and maintainability.
- **Native Experience:** 100% SwiftUI with adaptive layouts for various iPhone models.

## 🛠 Tech Stack
- **Language:** Swift 5.10
- **Framework:** SwiftUI
- **Architecture:** MVVM (Model-View-ViewModel)
- **Backend:** Firebase (Authentication, Firestore, Storage)
- **Dependency Manager:** Swift Package Manager (SPM)
- **Tools:** Xcode 15+, Git/GitHub
